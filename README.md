# Digital-IC-Learning
Digital IC learning 
1.Digital IC learning route
(1) 所需知识（以下几点）
①Verilog 语法
②数字电子技术基础
③笔试/面试 题目
④项目经历
⑤SV以及UVM
(2)数字基础部分
1.数制和码制
2.逻辑代数
3.门电路

5.时序逻辑：触发器、寄存器（移位）、计数器、脉冲发生器
6.存储器：rom、ram、SRAM、DRAM
(3)Verilog部分
1.语法、代码风格（2001）
2.各种状态机的写法
3.各种简单电路的代码片段
4.testbench的写法
(4)笔试题目部分
1.阻塞与非阻塞
2.同步与异步
3.建立时间与保持时间
4.竞争与冒险（亚稳态）
5.触发器和锁存器
